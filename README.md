# Github Notes

### Git
* Version Control: keeps track of the history and the things you changed on what you're working on.
* Collaboration: Help have the opportunity to have 2 people work on the same document at the same time.

### Github
* Web Service for Git: Lets you host Git repositories online.
* Provides a way to share projects, view histories, and collaborate with others.
* Access Anywhere: Your projects are stored in the cloud, so you can work from any device.

### Repository
* Another word can be "Repo"
* You can make projects
* You can have multiple files
* It has a README file describing the project (you have to manually select if you want one or not)
* It can be public or private

### Commit
* A save button
* Allows you to track changes you make
* Commit Message is a note describing what was changed in that version
* Commit Hash a unique identifier for each commit.
